# Telegram Mafia Bot

Production-ready Telegram bot for Mafia game with premium features.

## Features
- Complete Mafia game implementation
- Premium subscription system  
- YooKassa payment integration
- Advanced game roles and mechanics

## Deployment
- Python 3.11+
- Configured for Railway.app deployment
- Environment variables required (see env_example.txt)

## Files
- `mafia_bot.py` - Main bot code
- `premium_database.py` - Premium user database
- `Procfile` - Railway deployment config
- `requirements.txt` - Python dependencies
- `runtime.txt` - Python version spec
